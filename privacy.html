一、適用範圍

本隱私權政策適用於您使用以下相關服務時所涉及的個人資料蒐集、處理與利用：

  • 透過 LINE 官方帳號、LIFF 預訂表單進行的包棟預訂、諮詢與聯絡。
  • 透過本公司網站、表單頁面（如 LIFF 預訂頁、Google 表單等）填寫之預訂資料。
  • 其他由本公司主動提供，且連結本政策之網頁或服務。

若您由本服務連結至第三方網站（例如其他訂房平台、金流服務等）， 該第三方之隱私權政策由各該第三方自行負責，本公司之隱私權政策不適用於第三方網站。

二、我們蒐集的個人資料類型

為提供並改善本服務，我們可能蒐集下列類型的個人資料（實際蒐集項目以您填寫或授權為準）：

1. 識別類資料

  • LINE User ID（透過 LINE Login / LIFF 取得之使用者識別碼）。
  • 姓名或暱稱（包含您於 LINE 顯示的名稱、您在表單中填寫的姓名）。

2. 聯絡方式

  • 電子郵件信箱（包含您授權提供之 LINE 帳號 email，或您自行填寫之 email）。
  • 聯絡電話（若表單中有此欄位）。
  • 其他您主動提供的聯絡方式。

3. 預訂相關資料

  • 預訂館別、日期、天數。
  • 成人人數、兒童人數。
  • 是否攜帶寵物及相關說明。
  • 其他您於「需求內容」欄位填寫之資訊。

4. 使用紀錄與技術資料（視實際情形而定）

  • 使用本服務的時間、頁面瀏覽紀錄。
  • 您透過 LINE 與本公司官方帳號互動的紀錄（如訊息內容、按鈕點擊等）。

三、個人資料之蒐集方式

  • 您透過 LINE Login / LIFF 授權提供之資料（例如 LINE User ID、顯示名稱、email 等）。
  • 您在預訂表單中自行填寫之資料（姓名、聯絡方式、預訂內容等）。
  • 您與本公司聯繫時（例如 LINE 聊天、電話、電子郵件）主動提供之資訊。

我們僅在您同意提供或授權的範圍內蒐集資料。

四、個人資料利用之目的

我們會在下列目的範圍內使用您的個人資料，不會超出原本蒐集之特定目的：

  • 預訂服務之提供與履行：處理包棟預訂、查詢空房、金額試算、預訂確認等。
  • 通知與聯絡：發送預訂成功、變更、取消等通知，及與您聯繫訂單問題。
  • 客戶服務與關係維護：提供住宿相關資訊、注意事項與客服支援。
  • 營運與服務優化：統計分析預訂情形，以優化服務流程與使用者體驗。
  • 法令要求或權利行使：回應主管機關或司法機關合法之調查或要求，或為主張、行使及防禦法律上權利所必要者。

五、個人資料之利用期間、地區、對象及方式

1. 利用期間

自您提供資料之日起，至蒐集目的消失、您請求刪除或本服務終止營運為止（以先到者為準）。 若法律有特別保存期間之規定者，將依相關法令規定保存。

2. 利用地區

中華民國境內，以及本公司使用之雲端服務提供者伺服器所在地 （例如 Google 之伺服器可能位於境外）。

3. 利用對象

  • 本公司及受本公司委託處理業務之第三人（如雲端主機服務商、系統開發/維護廠商等）。
  • 為提供本服務所必要之合作對象（例如未來若導入之金流、訂房平台等）。
  • 依法有權要求機關或為保護本公司或第三人合法權益所必要者。

4. 利用方式

以自動化或非自動化方式，於合法及合理之範圍內蒐集、處理及利用您的個人資料，例如：

  • 儲存在 Google 試算表、資料庫或其他雲端服務。
  • 透過 Email、LINE 訊息、電話與您聯繫。
  • 用於統計分析，並在可行情況下，盡可能去識別化或匿名化。

六、第三人提供與國外傳輸

在下列情況下，我們可能將您的個人資料提供予第三人或進行境外傳輸：

  • 為提供本服務所必要（例如使用 Google 雲端服務、LINE 平台等）。
  • 為遵守法律或主管機關合法要求之必要。
  • 為保護您的權益或他人生命、身體、自由及財產安全所必要者。

除前述情形外，未經您同意，我們不會任意出售、交換、出租或以其他方式提供您的個人資料予任何第三方。

七、資料安全與保護措施

我們將以合理之技術及組織安全措施保護您的個人資料，防止未經授權之存取、洩漏、竄改或破壞，包括但不限於：

  • 限制有業務需要之人員才能存取相關資料。
  • 管理帳號與權限，避免未授權存取。
  • 定期檢視系統設定及安全機制。

然而，透過網際網路傳輸或電子儲存方式並無法保證百分之百安全，我們將盡力於合理安全水準內保護您的資料。

八、您的權利及行使方式

依個資法規定，您對於本公司所持有關於您的個人資料，得行使以下權利：

  • 查詢或請求閱覽。
  • 請求製給複製本。
  • 請求補充或更正。
  • 請求停止蒐集、處理或利用。
  • 請求刪除。

如您欲行使上述權利，請透過下列聯絡方式與我們聯繫。本公司將於確認您身分後，儘速處理您的申請； 惟法律另有規定或基於法定義務所必要者，得不依您的請求為之。

九、未成年人之保護

若您未滿法定成年年齡，應在法定代理人或監護人之同意與陪同下使用本服務， 並在其同意下提供個人資料。當您使用本服務時，即表示您的法定代理人或監護人已閱讀、了解並同意本政策。
